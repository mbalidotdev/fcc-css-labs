# Learning Notes

## What I Learned

### Styling Properties

- `border-radius` - for rounded corners on the card and task boxes
- `box-shadow` - to create depth and elevation effect on the main card
- `list-style: none` - to remove default bullet points from lists

### Custom Checkbox Styling

- Learned how to style the checkbox container to make it visually distinct
- Used background color and padding to create task "boxes"

### Pseudo-Classes for Link States

- `:hover` - changes link color on mouse hover
- `:active` - changes color when link is clicked
- `:focus` - adds outline when link is focused (keyboard navigation)
- `:visited` - changes color for previously visited links

## Interesting Discovery

I structured the entire to-do list using `<ul>` and `<li>` elements. It makes perfect sense since it's literally a "list" - both semantically and visually!

## Visual Appeal

This was one of my more visually appealing projects. The card design with shadows and rounded corners made it feel more polished and modern compared to basic layouts.
