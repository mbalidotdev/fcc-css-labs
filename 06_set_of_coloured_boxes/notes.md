# Learning Notes

## What I Learned

### Color Value Formats

This project pushed me out of my comfort zone by using different color value formats instead of just named colors:

- **Hexadecimal** - `#000051` (dark blue)
- **RGB** - `rgb(255, 0, 177)` (hot pink)
- **Named colors** - `green` (still used one!)
- **HSL** - `hsl(22, 100%, 57%)` (orange)
- **Radial gradient** - `radial-gradient(circle, red, blue)` (gradient effect)

### Additional Styling

- `box-shadow` - added depth to make boxes pop off the page
- `radial-gradient` - created a smooth color transition from center outward
- `border-radius` - rounded corners for a softer look

## Project Evolution

I first coded the boxes according to basic requirements, then enhanced them with shadows and gradients to make them more visually interesting. The enhanced version is what's in this project.

## Why This Was Fun

Getting comfortable with different color formats is essential for CSS. This simple project made practicing color values more engaging than just reading about them!
